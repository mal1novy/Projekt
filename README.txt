Program jest konsolową symulacją platformy streamingowej, umożliwiającej użytkownikowi przeglądanie katalogu treści (gry, filmy, seriale), ich zakup, wypożyczanie oraz korzystanie z subskrypcji czasowej. System działa z perspektywy użytkownika.

Funkcjonalności
wyświetlanie całego katalogu treści
filtrowanie treści według kategorii (Game / Movie / Series)
wyszukiwanie treści po fragmencie tytułu
zakup treści na stałe
wypożyczanie treści na określoną liczbę dni
zakup subskrypcji czasowej
doładowanie konta
uruchamianie/odtwarzanie treści z uwzględnieniem dostępu
symulacja upływu czasu (wygasanie subskrypcji i wypożyczeń)
zapis i odczyt danych użytkownika do/z pliku
